# How to run textbook html generator

install packages
- `npm install`

login to google account
- `npx clasp login`

create apps script project
- `npx clasp create --type webapp --title "TextbookGenerator" --rootDir ./dist`

build
- `npm run build`

push
- `npm run upload`
